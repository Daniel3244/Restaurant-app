# Restaurant-app

Aplikacja webowa do zarządzania restauracją.

## Moduły
- Panel klienta (przeglądanie menu, składanie zamówień)
- Panel pracownika (zarządzanie zamówieniami)
- Panel menedżera (zarządzanie menu, raporty)
- Ekran numerków (dla klientów)

## Technologie
- Frontend: React + TypeScript (Vite)
- Backend: Java + Spring Boot (wkrótce)
- Baza danych: MySQL

## Uruchomienie frontendu

```powershell
cd frontend
npm install
npm run dev
```
